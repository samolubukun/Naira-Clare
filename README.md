## ✨ What is NairaClare?

NairaClare transforms the way Nigerian freelancers, employees, and small business owners manage their money through **AI-powered intelligence and automated compliance**. Whether you're calculating your monthly PAYE, tracking side-income WHT credits, or preparing your annual FIRS returns, NairaClare provides real-time visibility into your tax health and financial growth.

### 🎯 **The Magic Behind NairaClare**

```
💸 Log Income/Expense
    ↓
🧠 AI Tax Engine (NTA 2025 Logic)
    ↓
📊 Live Tax Meter & Liability Gauge
    ↓
📄 Automated Invoicing & VAT
    ↓
🤖 AI Auditor & Assistant (Clare)
    ↓
📅 Compliance Calendar & Filing Summaries
```

Our intelligent **financial pipeline** creates comprehensive tax insights:
- 📊 **Live Tax Meter** → Instant calculation of liabilities and reliefs
- 🧠 **AI Auditor** → Identifies missing deductions and compliance risks
- 🧾 **AI Receipt Scanner** → Extracts expense data from photos with tax categorization
- 🏦 **Statement Import** → Parses PDF bank statements into organized logs
- 📅 **Smart Reminders** → Automated alerts for VAT, WHT, and annual filing deadlines

## 🚀 Quick Start

### 📋 **Prerequisites**
- **Node.js** 18+
- **Gemini API Key** (for scans and audit)
- **Convex Account** (for real-time backend)

### ⚡ **Installation**

```bash
# 1️⃣ Clone the repository
git clone https://github.com/samolubukun/nairaclare.git
cd nairaclare

# 2️⃣ Install dependencies
npm install

# 3️⃣ Start development
npm run dev
```
